# KernelSU_AutoBuild_Cepheus

使用GitHub Action定期自动构建内核并加入kernelsu。
由于kernel的更新，将来可能会编译失败。

支持小米9 ROM:
- ~~crDroid 9.2~~
- PixelExperience  12.1
- ~~EvolutionX~~
- PixelOS A13-A14

感谢@nozomanai提供的编译模板

支持PixelOS A13-A14和PixelExperiencePlus A12。

    由于crDroid内核的更新，不再适配KernelSU

所有内核均已加入安全启动模式（开机第一屏三击音量键）
